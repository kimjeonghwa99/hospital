# hospital

병원 ERP (Hospital ERP)  
현재 초기 개발 단계입니다.

---

## 📌 프로젝트 개요

본 프로젝트는 병원 운영에 필요한 기본 기능을 단계적으로 구현하는  
**웹 기반 병원 ERP 시스템**입니다.

성형외과를 포함한 병원 환경을 기준으로 설계되었으며,  
다양한 직군의 역할을 고려하여 확장 가능하게 개발 중입니다.

---

## 👥 주요 사용자(직군)

본 ERP는 다음과 같은 병원 직군을 기준으로 설계됩니다.

- **의사**
  - 진료 확인
  - 환자 정보 조회
  - 향후 진료/상담 기록 관리

- **간호사**
  - 진료 보조
  - 환자 상태 확인
  - 근무/근태 관리

- **상담실장**
  - 환자 상담 관리
  - 예약 및 상담 일정 확인
  - (성형외과 등 특화 병원 환경 고려)

- **관리자**
  - 사용자 및 권한 관리
  - 병원 설정 관리
  - 근태/정책 관리

> 병원 유형(성형외과, 일반 병원 등)에 따라  
> 직군 및 기능은 유연하게 확장될 예정입니다.
